# Biometric Authentication Demo - Flutter

A comprehensive demonstration app showcasing biometric authentication capabilities in Flutter using the `local_auth` package. This app demonstrates on-device biometric authentication (Face ID, Touch ID, and fingerprint) for both iOS and Android platforms, ensuring secure and seamless user authentication without requiring passwords.

This is companion code repository for article: <a href="https://mobisoftinfotech.com/resources/flutter-development/secure-flutter-biometric-authentication-fingerprint-face-id">Secure Biometric Authentication in Flutter: Fingerprint & Face ID Login Tutorial</a> by <a href="https://mobisoftinfotech.com/">Mobisoft - App Development Company, Houston</a>
